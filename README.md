# shwxnbookstore

Full-stack online bookstore — Next.js 14 + Express + Firestore + Cloudinary.

See **[SETUP.md](SETUP.md)** for step-by-step installation, environment setup, seeding, and running dev servers.

```
shwxnbookstore/
├── frontend/   # Next.js 14 (App Router, TS, Tailwind)
└── backend/    # Express REST API + Firebase Admin + Cloudinary
```

Default admin (after seed): `admin` / `admin123`.
